# CarNotGoat
## Install dependencies
- [Keras][a69fecb3]
- [TensorFlow][b8b3f1f6]
```bash
sudo -H pip3 install tensorflow keras
```

  [a69fecb3]: https://keras.io/#installation "Install keras"
  [b8b3f1f6]: https://www.tensorflow.org/install/install_sources "Install tensor flow"
## Run
```bash
./monty > shows.csv
./onehot
./learn
```
